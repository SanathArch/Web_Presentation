# 📐 Obsidian Templates for Tropical Architecture Initiative

## How to Use These Templates

1. **Open Obsidian** with this project folder as your vault
2. **Install the "Templater" plugin** (optional but recommended)
   - Go to Settings → Community Plugins → Browse
   - Search for "Templater" and install
   - Set this folder as your templates location

3. **Create New Note from Template:**
   - Right-click in file explorer → "Create new note from template"
   - OR use keyboard shortcut (configure in settings)
   - OR copy template and paste into new note

## 🔗 Graph View Optimization

All templates include:
- **Internal links** to `[[project-memory]]` and relevant pages
- **Consistent tags** for filtering in graph view
- **Frontmatter** with metadata for Dataview plugin

### Recommended Graph Settings:
1. **Groups by Tag:**
   - `#meeting` - Blue
   - `#design` - Orange
   - `#research` - Green
   - `#product` - Purple
   - `#sprint` - Red

2. **Filters:**
   - Show: `path:includes("templates")` to see templates
   - Hide: `path:includes("templates")` to hide templates from main view

3. **Display:**
   - Arrow labels: On
   - Node labels: On
   - Line width: 2px

## 🔌 Recommended Obsidian Plugins

1. **Templater** - Auto-fill template variables
2. **Dataview** - Query and display notes dynamically
3. **Calendar** - Track notes by date
4. **Tasks** - Manage checkboxes across notes
5. **Tag Wrangler** - Organize and rename tags
6. **Outliner** - Better bullet point management

## 📝 Naming Conventions

**Notes should be named:**
- `YYYY-MM-DD Meeting with [Name/Topic]`
- `Design Decision - [Topic]`
- `Research - [Paper Title/Topic]`
- `Product - [Product Name] - Dev Log`
- `Case Study - [Project Name]`
- `Sprint [Number] Planning`

This ensures alphabetical and chronological organization!
